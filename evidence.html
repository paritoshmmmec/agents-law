The actual distinction

RBAC decides if the agent may touch the tool at all.
Evidence Gate decides if this call, right now, is justified.

Most agent stacks stop at the first question. That gap — between "allowed in general" and "justified in this instance" — is where agents take actions no one can later explain.

RBAC / IAM

Authorized

The agent has permission to call this tool. That permission was granted once, ahead of time, and doesn't know anything about the facts of this particular call.

Evidence Gate

Justified

This exact call is backed by evidence that is fresh, came from a source the agent was allowed to use, and doesn't conflict with anything else on record — checked at the moment it matters.

NORTH STAR
"Can we prove why this agent took this action, using facts it was actually allowed to rely on?"
If the answer is no — block, restrict, or route to review. Before harm occurs, not after.
